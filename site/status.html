Where We Stand

Target: Qwen 3.5 27B — 64 hybrid layers, GPTQ W4A16, GH200

Zero Forth. Zero Python. Zero CUDA runtime. All .ls source.

Architecture (confirmed): Two cooperative megakernels — one cubin per layer type. 48 DeltaNet layers in one cubin, 16 full-attention layers in another. Within each megakernel, layers loop at runtime with a cooperative grid-sync barrier (bar.sync / membar.gl) between iterations. Not compile-time unrolling — that would copy the layer body 48× or 16×, producing enormous inflexible code. Both cubins emitted as raw Hopper binary (direct instruction encoding). No SASS assembler, no ptxas, no PTX, no CUDA runtime.
